androidアプリの提供元を調べるために、PackageManagerのgetInstallerPackageNameを使えます

PackageManager pm = getPackageManager();

String installationSource = pm.getInstallerPackageName(getPackageName());

String source ;


if(installationSource.equals("com.android.vending")){

 source= "google play";

}else if(installationSource.equals("com.amazon.venezia")){

 source= "amazon market";

}else{

 source= "その他のマーケット";

}


Log.d("DEBUG", source + "からインストールされましたね。");

installationSourceの値は【com.android.vending】でしたら、google playからインストールされたってことです！
