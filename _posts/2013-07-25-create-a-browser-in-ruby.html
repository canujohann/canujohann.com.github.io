今日、rubysのソケット(socket)ライブラリーを使って、シンプルなブラウザーを作ってみましょう。

ライブラリーをインポートし、アクセスするサーバーの値をインスタンスに代入

require 'socket'
host = 'canujohann.com' # ウエブサーバー
port = 80 # サーバーのポート
path = '/index.html' # アクセスするページ

リクエストの設定:


#HTTPリクエスト
request = "GET #{path} HTTP/1.0\r\n\r\n"

実際に接続して、結果をプリントします。


socket = TCPSocket.open(host,port) # サーバー接続
socket.print(request) # 情報送信
response = socket.read # レスポンスの読み取り
headers,body = response.split("\r\n\r\n", 2) #body+headeを取得
print body # bodyを表示
