windows環境でLaravelフレームワークをインストールしましょう！まずは環境設定を行います。今回は3つのプログラムを使います：

ツール名 ダウンロードURL
XAMPP https://www.apachefriends.org/jp/index.html
GIT http://msysgit.github.io/
Composer https://getcomposer.org/doc/00-intro.md#installation-windows

上記プログラムのインストールが終わりましたら・・

1- GITでxamppのhtdocs直下でlaravelの最新版を落とします

cd C:\xampp\htdocs
git clone git@github.com:laravel/laravel.git laraveldev

C:\xampp\htdocs\laraveldev　フォルダーが作られました！

2- composerで必要なライブラリーをダウンロード

cd laraveldev
composer install

3- hostsファイルを編集　（C:\windows\system32\drivers\etc\hosts）

//この一行を追加
127.0.0.1             www.laravel.dev

4- VirtualHostの設定 (c:\xampp\apache\conf\extra\httpd-vhosts.conf)

    DocumentRoot "C:/xampp/htdocs/laraveldev/public"
    ServerName www.laravel.dev
    ServerAlias www.laravel.dev
    ErrorLog "logs/laravel.log"
    CustomLog "logs/custom.laravel.log" combined
    <Directory "C:/xampp/htdocs/laraveldev/public">
        AllowOverride All
        Order Allow,Deny
        Allow from all
        Require all granted

5- XAMPPでapacheを再起動

"http://www.laravel.dev/"をアクセスすると、新しいlarabelアプリが表示されます！おめでとうございます～
