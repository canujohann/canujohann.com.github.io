今日はCSS3のアニメーションを使ってprogress bar的なものを作ってみましょう！

まずはHTMLのソース：

[html]<span class="bar half-full"></span> [/html]

CSS側はこんな感じにします：

[css]

/* バー*/
.bar {
height:18px;
width:300px;
margin:2px 2px;
border:solid 1px #b5b5b5;
}

/*アニメーションクラス*/
.half-full {
/*infinite (無限ループ)でアニメーションを繰り返す　*/
-moz-animation: half-full 5s ease-out 0 infinite normal;
-webkit-animation: half-full 5s ease 0 infinite normal;
background-color: #ff3e36;
}

/* mozilla用 */
@-moz-keyframes half-full {0% { width:0px;} 100%{ width:90%;} }

/* animation for webkit (chrome,...) */
@-webkit-keyframes half-full { 0% { width:0px;} 100%{ width:90%;} }

[/css]

コメント通りですが、アニメーションを使ってバーを赤色で染めます（9割まで）。

もちろん、IEは未対応です！
