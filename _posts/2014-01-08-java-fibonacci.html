javaで【フィボナッチ数】を実装してみました。0からnまでループを回すと簡単にできますが、recursiveメッソードでやってみましょう！

public static Long fibonacci (int i){

if (i==0 || i==1) return (long)i;

return fibonacci(i-1) + fibonacci (i-2) ;

 }

問題ないですが・・・数字が大きければ、処理スピードが遅くなりますので、HashMapを追加しました。

public static HashMap<Integer,Long> val = new HashMap<Integer,Long>();


public static Long fibonacci (int i){

if (i==0 || i==1) return (long)i;

     if (val.get(i) != null) return val.get(i);

     val.put(i,fibonacci(i-1) + fibonacci (i-2));

     return val.get(i);

 }
